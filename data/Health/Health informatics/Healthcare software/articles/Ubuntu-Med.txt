The Ubuntu-Med project  is a Kubuntu-based operating system customization created to provide a co-ordinated operating system and collection of available free software packages that are suited to the requirements of medical practices and research. Ubuntu-Med includes packages in the categories: All software and packages are open-source, with a goal of GPL-licensure. This system has been in use by some medical practitioners.  Ubuntu-Med is available as a self-installing Live CD image (which is a Kubuntu remaster) or as a Filesystem Archive. Both are available from the project website at Sourceforge.

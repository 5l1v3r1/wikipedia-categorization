IP2Location is a Malaysian company offering IP geolocation software applications, i.e. tools that attempt to derive geographical data (country, region, city, latitude, longitude, ZIP code, time zone), and also connection speed, ISP and domain name, about an Internet user using their IP addresses.  IP2Location.com is a subsidiary of Hexasoft Development Sdn. Bhd. ("HDSB"), a company based in Penang.  The software can be integrated with database management systems such as MySQL, SQL Server or others. It comes with API in several programming languages .NET,  Java, C, C++, Visual Basic, PHP, ColdFusion,  Perl, Ruby and Python. IP2Location has been used by many other applications such as Wikiscanner.

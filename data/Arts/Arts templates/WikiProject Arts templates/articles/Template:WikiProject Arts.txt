This WikiProject banner uses {{WPBannerMeta}}, a meta-template for easily creating and maintaining banners and talk-page notices. Please report any such errors or feature requests to Template talk:WPBannerMeta. Any information or warnings about the template are below: To use this template, add {{WikiProject Arts}} to the talk page of any article that falls within the scope of the WikiProject Arts. Please do not subst:.  The template uses a few additional attributes. cat = yes, if applying the template to a category page. Otherwise, leave blank.   class = FA, A, GA, B, C, Start or Stub. If left blank, class will appear as question marks. It is unnecessary to set class for category pages. See here for guidance.  The VA project has decided not to use importance ratings.
